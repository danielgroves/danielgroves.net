Skills

As a developer I like to keep work agile. I use Git for version control, making it easy to collaborate with large teams of other developers. I can write unit tests for applications, which allows for easy automated testing of software to ensure that changes in the future to not break the application. When developing larger application I prefer to use a web framework, as this provides the scaffolding for the project and saves on development time by not reinventing the wheel.

When designing I work directly in the browser. This allows designs to be developed quickly whilst showing the full dimensions of a design from desktop all the way through to mobile, something missed by graphical representations. In the past I have worked in Adobe Fireworks and Photoshop to produce site mockups and Adobe Dreamweaver to produce finished products, although I prefer to hand-code sites as it provides more flexibility and more options for incorporating new technologies.
